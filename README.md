# HR Analytics Dashboard (Power BI)

## 📊 Project Overview
This project is an interactive HR Analytics Dashboard built using Power BI to analyze employee salary and workforce distribution.

## 🔧 Tools Used
- Power BI
- DAX
- Excel (for dataset)

## 📌 Key Features
- KPIs: Total Salary, Average Salary, Min & Max Salary, Employee Count
- Department-wise employee distribution
- Salary trend analysis over years
- Filters for country and city
